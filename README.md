#  Gerador de Currículos - UNIPAR

Sistema web desenvolvido em PHP para criação e geração de currículos profissionais.

##  Tecnologias Utilizadas

- **PHP** - Backend e processamento de dados
- **HTML5** - Estrutura das páginas
- **CSS3** + **Bootstrap** - Estilização e design responsivo
- **JavaScript** - Interatividade e campos dinâmicos
- **Git** - Controle de versão

##  Funcionalidades

- ✅ Formulário dinâmico com campos expansíveis
- ✅ Cálculo automático de idade
- ✅ Adição de múltiplas formações acadêmicas
- ✅ Adição de múltiplas experiências profissionais
- ✅ Geração de currículo formatado para impressão
- ✅ Design responsivo com Bootstrap
- ✅ Interface intuitiva e moderna

##  Estrutura do Projeto
gerador-curriculo-unipar/
├── index.php # Página inicial
├── formulario.php # Formulário de dados
├── curriculo.php # Geração do currículo
